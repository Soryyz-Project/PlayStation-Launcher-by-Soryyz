# Contributing to SLauncher

Thanks for your interest! Here's how you can help.

## Issues

- Use the issue templates (bug report / feature request)
- Search existing issues before creating a new one
- Provide clear steps to reproduce for bugs

## Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run `npm run tauri build` to verify it compiles
5. Open a PR with a clear description of what and why

## Code Style

- No unnecessary comments
- Follow existing patterns in the codebase
- Keep components focused and hooks lightweight

## Questions

Open a discussion or issue — we're friendly.
